# Suicide Attempts in Poland: Power BI Dashboard (2017–2024)

## Overview

This project is an interactive **Power BI** report that presents official statistics on suicide attempts in Poland.  
The dashboard provides insights into:

- **Trends over time** – yearly and monthly changes  
- **Reasons and circumstances** behind suicide attempts  
- **Age groups and gender differences**  
- **Social and employment status** of individuals  
- **Regional distribution** of cases across Poland  
- **Clinics and mental health facilities** available in each voivodeship, where people in need may seek support  

The report is designed with a clear and respectful visual style appropriate for a sensitive public health topic.

---

## Data

- **Source:** Official data from the **Polish Police (Komenda Główna Policji)**  
- **Geographical scope:** Poland (national and regional level)  
- **Time range:** **2017–2024**

---

## Technology Used

- **Power BI Desktop**  
- Custom visualizations including shape maps, drill interactions, and dynamic filtering  
- Clean, muted color palette to ensure clarity and sensitivity

---

## Purpose

The goal of this project is to provide a structured, data-driven overview of suicide attempt patterns in Poland and highlight available support facilities.  
It is intended for portfolio presentation and educational analysis.
